My Story

From debugging native crashes to designing cloud pipelines

I joined Microsoft as an individual contributor on the Android team for Word, Excel, and PowerPoint. I wrote shared C++ and Java code that ran across all three apps ‚Äî multi-threaded rendering, File I/O, print subsystems, licensing, boot performance optimization, and file open scenarios. When something broke at the native layer, I was the one debugging JNI bridges and C++ stacktraces at the lowest levels.

Then came the ambitious project to unify all three apps into a single Office Mobile application. We launched first in China with a lean team, proved the concept, got funded, grew the team, and deployed it worldwide. I experienced firsthand what it takes to converge divergent codebases into a single product at global scale.

After that, I was part of the founding team that built Microsoft Viva Learning from an empty repo to a shipped product ‚Äî using React and modern full-stack architecture to create a learning platform now used across enterprises worldwide.

Today, I lead the ODSP Storage team at Microsoft ‚Äî blob storage, critical data migration pipelines, and governance systems for enterprise customers who need their data secure, compliant, and available at scale.
